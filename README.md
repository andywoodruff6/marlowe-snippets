# Marlowe DSL VSCode Snippets

![Marlowe Image](images/Marlowe_0.png) 

## Description

These are a collection of snippets to make it easier and faster to create Marlowe contracts.

## Snippets

The current snippets are:

- Assert
- Bound
- Case
- Choice
- Deposit
- If
- Let
- Notify
- Party
- Pay
- Payee
- Value
- When
